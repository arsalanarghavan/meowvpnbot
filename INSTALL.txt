# 🚀 دستور نصب

## نصب از GitHub:

```bash
git clone https://github.com/yourusername/meowvpnbot.git && cd meowvpnbot && sudo ./install.sh
```

## یا به صورت جداگانه:

```bash
# 1. دانلود
git clone https://github.com/yourusername/meowvpnbot.git

# 2. ورود به پوشه
cd meowvpnbot

# 3. نصب
sudo ./install.sh
```

---

## پس از نصب:

Setup Wizard خودکار باز می‌شود:
```
https://dashboard.yourdomain.com/setup
```

---

## به‌روزرسانی:

```bash
cd /var/www/meowvpnbot
sudo ./update.sh
```

---

## حذف کامل:

```bash
cd /var/www/meowvpnbot
sudo ./uninstall.sh
```

---

## مشکل داری؟

```bash
# رفع مشکلات Laravel
cd /var/www/meowvpnbot
bash fix_laravel.sh

# بررسی Nginx
bash check_nginx.sh

# دیباگ 404
bash debug_404.sh
```
