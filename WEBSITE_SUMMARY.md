## 🚀 نحوه استفاده

### نصب و راه‌اندازی:

```bash
# 1. نصب dependencies
cd site
composer install

# 2. تنظیم .env
php artisan key:generate

# 3. اجرا
php artisan serve
```

### دسترسی به پنل:
- URL: `http://localhost:8000`
- صفحه اول به داشبورد redirect می‌شود
